## 更换开机动画图片

* 替换开机动画图片

```
/usr/share/plymouth/ubuntu-logo.png
/usr/share/plymouth/themes/spinner/watermark.png
/usr/share/plymouth/themes/spinner/bgrt-fallback.png
```

> 将自定义图片替换到以上三处路径
>
> 注意图片格式为PNG



* 更新开机动画图片

```
$ sudo update-initramfs -u
$ reboot
```
